    Server

    整个服务端类。这里初始化了整个项目传递各种对象的global对象。

    global对象包含以下几个字段：

    1. config：项目的配置
    2. db、redis：Mongoose链接和Redis链接
    3. sio：Socket.IO对象
    4. email：nodemailer对象

    对于Koa的中间件来讲，这个对象可通过ctx.global获得。
